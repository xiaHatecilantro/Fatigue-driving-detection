# 实验记录

## 实验 1：MobileNetV3 → YOLO11m ROI 模型

| 项目 | 值 |
|------|------|
| 日期 | 2026-05-31 |
| 目的 | 将分类模型从 MobileNetV3 迁移到 YOLO11m，验证兼容性 |
| 数据 | `unified_dataset_roi_v1`（2000 张人脸裁剪，关键词归类） |
| 模型 | yolo11m-cls（预训练 ImageNet） |
| 训练参数 | 12 epoch, lr=0.0005, batch=32, imgsz=224 |
| 训练环境 | RTX 4060 8GB |
| 训练耗时 | ~8 分钟 |
| 验证准确率 | **96.8%** |
| 正常 F1 | 96.0% |
| 闭眼 F1 | 99.5% |
| 哈欠 F1 | 100% |
| 分心 F1 | 92.6% |
| 结论 | 模型迁移成功，指标接近原 MobileNetV3（97.8%），ROI 裁剪后人脸分类效果良好 |
| 问题 | 后续交叉验证发现 yawn 类数据存在系统性标注错误——文件夹名为 yawn 但图片中嘴巴闭合 |

## 实验 2：新数据集 YOLO11m 全图分类

| 项目 | 值 |
|------|------|
| 日期 | 2026-05-31 |
| 目的 | 替换数据集为可信标注，支持全景图直接分类（无需 MediaPipe ROI 裁剪） |
| 数据 | `yolo_fullframe`（7800 张全景驾驶图，人工标注，来源 IEEE ITSC 2025） |
| 模型 | yolo11m-cls（预训练 ImageNet） |
| 训练参数 | 15 epoch, lr=0.0005, batch=32, imgsz=224 |
| 训练环境 | RTX 4060 8GB |
| 训练耗时 | ~39 分钟（Early stopping at epoch 10） |
| 验证准确率 | **94.3%** |
| 正常 F1 | 96.1% |
| 闭眼 F1 | 93.3% |
| 哈欠 F1 | 87.6% |
| 分心 F1 | 79.5% |
| 结论 | 准确率略低于旧模型，但数据可信、标注可靠。模型支持全景图直接推理，可独立导出部署。分心类 recall 偏低，后续可补充分心数据 |

---

## 对比总结

| 指标 | 实验 1（旧数据 ROI） | 实验 2（新数据全景） |
|------|---------------------|---------------------|
| 训练量 | 2000 张 | **7800 张（3.9x）** |
| 数据标注 | 关键词自动归类 | **人工标注** |
| 标注可信度 | yawn 类不可信 | **全部可信** |
| 准确率 | 96.8%* | **94.3%** |
| 推理方式 | 需 MediaPipe ROI 裁剪 | **全景直接推理** |
| 独立部署 | 依赖 MediaPipe | **✅ best.pt 即用** |
| 训练代码量 | 手写 350 行 | **ultralytics API 80 行** |
